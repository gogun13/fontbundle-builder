# Fontbundle Generator (Alpine 3.18)

## How to run (Docker / Podman):

```
chmod +x generate-fontbundle.sh
./generate-fontbundle.sh 3.18
```

Output:
- `fontbundle/`
- `fontbundle.tar.gz`

## Place your Loma fonts in:

```
src/main/resources/fonts/
```

Required files:
- Loma.ttf
- LomaBold.ttf
- LomaOblique.ttf
- LomaBoldOblique.ttf
